This plugin shows a screensaver on your website when there is no activity for some time.

== Description ==

This plugin shows a screensaver on your website when there is no activity for some time. It have 5 screensavers that can choosen to something funny for your visitors when they leave their screen for a long time.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
